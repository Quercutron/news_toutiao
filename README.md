<h1>AI掘金头条新闻系统(Toutiao News)</h1>

一个基于FastAPI和SQLAlchemy构建的现代化新闻系统,支持用户注册登录、新闻浏览、收藏和历史
记录等功能。

<h2>1-1 项目概述</h2>

这是一个仿今日头条的新闻系统后端API服务,采用异步Python框架FastAPI开发,使用MySQL作为数据
库存储,通过SQLAlchemy ORM进行数据访问。系统提供完整的用户管理、新闻浏览、收藏和历史记录
功能。

<h2>1-2 技术栈</h2>

· 后端框架:FastAPI

· 数据库:MySQL

· ORM:SQLAlchemy(异步)

· 数据库驱动:aiomysql

· 密码加密:passlib+bcrypt

· 缓存系统:Redis

· 异步支持:Python asyncio

<h2>1-3 功能模块</h2>

· 1.后端框架:FastAPI

· 2.用户管理模块

· 3.新闻模块

· 4.收藏模块

· 5.缓存模块
